# nora-gyazo

## What is this

- Electronで実装された、オープンソースなGyazo用の常駐アプリです
- 以下の機能が実装される予定です
  - Downloadsフォルダ内の全画像をGyazoにアップロード
  - Picturesフォルダ内の全画像をGyazoにアップロード
  - 1分おきに全画面キャプチャしてGyazoにアップロード


- おもしろいアイデアがあったらPull Requestを送って勝手に追加してください
  - 採用された場合、Gyazo Pro 1年分をあげます


## Install

```
npm install
```


## start

```
npm start
```
